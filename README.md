# eXMB
Bot to mirror r/formula1 highlight clips, powered by [eXRC](https://github.com/eXhumer/pyeXRC) & [eXVHP](https://github.com/eXhumer/pyeXVHP).

## Package Installation Guide
### Requires
* Python 3.9+
* pip package manager

### Install from source
```console
pip install .
```
