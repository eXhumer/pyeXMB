# eXMB
Bot to mirror supported video links from a subreddit, powered by [eXRC](https://github.com/eXhumer/pyeXRC) & [eXVHP](https://github.com/eXhumer/pyeXVHP).

## Package Installation Guide
### Requires
* Python 3.9+
* pip package manager

### Install from source
```console
pip install .
```
